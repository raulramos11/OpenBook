# OpenBook

OpenBook e um esboco de aplicacao web para busca e organizacao de livros. O projeto apresenta um catalogo inicial com titulo, autor, tema, resumo, fonte e disponibilidade, permitindo pesquisar por palavras-chave e filtrar obras entre gratuitas, pagas ou a verificar.

## Tecnologias

- Java 21
- Spring Boot
- Gradle
- HTML, CSS e JavaScript
